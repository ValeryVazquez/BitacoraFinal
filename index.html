¿Qué son las etiquetas HTML?

Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo. Son el lenguaje utilizado para estructurar y definir el contenido en un documento HTML. Cada etiqueta contiene instrucciones sencillas que indican al navegador cómo dar formato al texto y a definir los diversos elementos de la página web

¿Cómo se compone una etiqueta en HTML?

  • Una etiqueta de apertura — esta empezará con un símbolo < >.
  • Contenido — unas breves instrucciones sobre cómo mostrar el elemento en la página.
  • Una etiqueta de cierre — cerrará con un símbolo < / >.

    • Etiquetas Basicas

    1. (!DOCTYPE) -Se utiliza para especificar la versión de HTML que se está utilizando en la página.
    2. (html) -Define el inicio y el final de la página web.
    3. (head) -Define la sección de encabezado de la página, donde se incluyen elementos como el título de la página, metaetiquetas, scripts, entre otros.
    4. (title) -Define el título de la página web que aparece en la pestaña del navegador.
    5. (body) -Define la sección del cuerpo de la página web, donde se incluyen todos los elementos que se mostrarán en la página.
    6. (h1) a (h6) -Define los encabezados o títulos de diferentes niveles de jerarquía en la página web. El uso de esta etiqueta formateará cualquier texto entre la etiqueta (h) de apertura y la etiqueta (/h) de cierre como un Título o subtítulo.
    7. (style) -Se usa para definir cómo se ven los elementos de una página web.
    8. (img) -Define una imagen que se mostrará en la página web.
    9. (p)-Etiqueta de párrafo(/p): Define un párrafo de texto. Las etiquetas (p) y (/p) son etiquetas HTML y la "Etiqueta de párrafo" es el elemento HTML, es decir, el texto de la página.
    10. (video) -El elemento (video) se utiliza para incrustar vídeos en un documento HTML o XHTML.
    11. (hover) - permite aplicar efectos a un elemento cuando el usuario pasa el cursor sobre él.

      12. Remplazar los ( ) Por < > para que el codigo funcione
